# FarmLink

## Build Setup

```bash
# install dependencies for Server and Client
$ npm run install

##########################

#Serup server API
$ cd Server

# Add Client.env
$ echo "FULLURI = "{YOUR_URI}" " >> Client.env

# OR
$ echo "USERNAME="{YOUR_USERNAME}"" >> Client.env
$ echo "PASSWORD="{YOUR_PASSWORD}"" >> Client.env

##########################

# run Server:5050 and Client
$ npm run dev
```
